# RoundFi — Project Status

**Updated:** 2026-05-07 · **Tracking branch:** `main` · **Live:** [roundfinancial.vercel.app](https://roundfinancial.vercel.app) · **Devnet:** [4 programs · protocol initialized · pool seeded · 3 members joined](./devnet-deployment.md)

> **TL;DR for evaluators / partners.** RoundFi is a multi-track project: a polished **off-chain product** (front-end + L1 actuarial simulator) is live and demoable today, and the **on-chain Anchor programs are deployed, initialized, AND running their first fully-populated ROSCA pool on Solana Devnet** — `ProtocolConfig` + `ReputationConfig` + `Pool` PDAs (with all 4 USDC vault ATAs) plus 3 `Member` PDAs (with Metaplex Core position NFTs minted and $45 USDC escrowed) hold real state, verifiable via Solscan ([deployment register](./devnet-deployment.md)). `anchor build` is a required CI gate on `main`; the full bankrun integration suite is the next layer of verification. The B2B oracle and production indexer are explicitly **roadmap (Phase 3 / post-grant)**. This doc is the authoritative status register — anything claimed elsewhere should reduce to one of the three columns below.

## Recent polish wave (PRs #140–#156)

The cutover before the grant video closed a long tail of UX, a11y, and integration gaps. Summarised so the recent diff is legible without scrolling the GitHub timeline:

- **Toolchain + CI** — `anchor build` flipped to required (#140); workspace `tsc` gated (#142); prettier sweep + lint gate restored (#143); branch protection enforces the 3 required lanes (#146); audit-bankrun gap honestly documented (#144).
- **System pages** — branded `not-found.tsx` (#147), `loading.tsx` (#148), and `error.tsx` (#149) with the same neon-on-deep-navy palette as the dashboard.
- **Feedback layer** — `sonner` toasts on every user-initiated session event (#150); modal loading states (`Processando…` + disabled CTAs during the 900–1500ms confirmation window) on BuyOffer / SellPosition / Send / WithdrawYield / PayInstallment (#151).
- **A11y** — focus trap + focus restore + autofocus on `Modal.tsx` (#135); focus-visible ring + `aria-label` / `aria-valuetext` on form inputs / sliders + close-button localized to PT (#152).
- **Live tier mechanics** — `LEVEL_TABLE` drives `user.level` / `levelLabel` / `nextLevel` / `colateralPct` / `leverageX` from `user.score`; `PAY_INSTALLMENT` and `LOAD_FROM_DEMO` recompute on every score change; cross-tier crossings emit a celebratory `levelup` event + 6s toast; `SideNav` flips to a purple-teal "✦ NÍVEL MÁXIMO" gradient at lvl 3; `LevelsList` ladder + `ReputationPassport` pill + home colat KPI all flip on the same flag (#153, #154).
- **Demo Studio bridge** — `LOAD_FROM_DEMO` now ships a synthesized `ActiveGroup` (carta + currentMonth + contemplated state); `FeaturedGroup` consumes it so `/home` reflects the chosen preset instead of the default fixture (#154).
- **Pay-installment integrity** — modal blocks submit when cycle is fully funded or balance < installment; reducer carries matching defensive guards; dial caps at `total` cleanly (no more month-8 ceiling on a 12-cycle group); shipped within #153.
- **Audit error path coverage** — `tests/security_audit_paths.spec.ts` adds one canonical failing-path case per error code from #122–#127 (4 implementable; 2 documented as guard-only post-CPI defenses unreachable on real `mpl-core`) (#155).
- **Secondary market integration** — `listings` + `acquiredPositions` moved into `SessionState`; `SellShareModal` (`/carteira`) and `SellPositionModal` (`/mercado`) write to the same listing slice; `PositionsList` renders a `◆ LISTADO` badge for listed cotas; `BuyOfferModal` mints an `NftPosition` into `acquiredPositions` so a bought cota immediately renders in `/carteira` (#156).

- **What's roadmap:** column 3. We are explicit about M3 (app ↔ on-chain wiring), Phase 2 (indexer), and Phase 3 (B2B oracle, mainnet). These are not implied features.

If anything in any other doc / pitch / pull request implies that an item from column 3 is shipped, that's a doc bug — please file or correct against this status register.
