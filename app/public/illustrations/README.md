## Format

- **Format**: PNG (transparent or solid dark background works — cards already have a tonal gradient behind)
- **Aspect**: roughly square (1:1) — cards reserve a `aspect-square` slot
- **Min width**: 800px (cards render up to ~480px wide on desktop with retina)
